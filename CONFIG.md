## 🎛️ Configurable Parameters

### OpenAI Configuration

```python
# Model to use
OPENAI_MODEL = "gpt-4o-mini"  # Options: gpt-4, gpt-4-turbo, gpt-3.5-turbo
# Note: gpt-4o-mini is recommended for performance/cost

# Temperature (model creativity)
LLM_TEMPERATURE = 0.7  # Range: 0.0 (deterministic) - 1.0 (creative)
# Recommendations:
#   - 0.3-0.5 for precise tasks
#   - 0.7 for natural summaries
#   - 0.9+ for creative generation

# Max tokens per response
LLM_MAX_TOKENS = 1000  # Increase for longer summaries
```

### Tavily Configuration

```python
# Maximum number of search results
TAVILY_MAX_RESULTS = 5  # Range: 1-10
# Notes:
#   - 5 is balance between quality and variety
#   - +10 can add noise
#   - -3 can leave information gaps

# Include response generated by Tavily
TAVILY_INCLUDE_ANSWER = False  # True if you want Tavily's summary response
```

### System Configuration

```python
# Output language
SYSTEM_LANGUAGE = "en"  # "es" (Spanish), "en" (English), etc.

# Verbose mode (more logs)
VERBOSE_MODE = True  # True for debug, False for production

# Show emojis in output
USE_EMOJIS = True  # True, False

# Timeout for API calls (seconds)
API_TIMEOUT = 30  # Increase for slow connections
```

### Quiz Configuration

```python
# Number of summary paragraphs
```

## 🔧 Parameter Modification

### Option 1: Environment Variables (.env)

```bash

OPENAI_MODEL=gpt-4-turbo
LLM_TEMPERATURE=0.5
TAVILY_MAX_RESULTS=3
VERBOSE_MODE=True
```

### Option 2: Code (in the notebook)

Before executing `run_healthbot()`, redefine:

```python
# Customize configuration
OPENAI_MODEL = "gpt-4-turbo"
LLM_TEMPERATURE = 0.5
TAVILY_MAX_RESULTS = 3

# Reinitialize tools
llm = ChatOpenAI(model=OPENAI_MODEL, temperature=LLM_TEMPERATURE)
search_tool = TavilySearchResults(max_results=TAVILY_MAX_RESULTS)

# Then execute
run_healthbot()
```

## 📈 Performance Metrics

To track performance:

```python
import time

class PerformanceMetrics:
    def __init__(self):
        self.start_time = None
        self.end_time = None
        self.search_time = 0
        self.summary_time = 0
        self.question_time = 0
        self.grade_time = 0
    
    @property
    def total_time(self):
        return self.end_time - self.start_time
    
    def print_report(self):
        print(f"⏱️ Search: {self.search_time:.2f}s")
        print(f"⏱️ Summary: {self.summary_time:.2f}s")
        print(f"⏱️ Question: {self.question_time:.2f}s")
        print(f"⏱️ Grading: {self.grade_time:.2f}s")
        print(f"⏱️ TOTAL: {self.total_time:.2f}s")

# metrics = PerformanceMetrics()
```
